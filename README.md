# TicTacToe
A TicTacToe game, made in python, pygame. It works over a network using sockets.

In order to start a game, initiate a server by opening the 'server.bat' file. 

Once the server is up and running, ensure all 'config.json' files for the 
clients are corresponding to the one created in the folder of the server. 

To connect to this server, simply open the 'client.bat' file. 

Each batch file will show a console window for debugging purposes that 
will be hidden once the application gets started. When the application 
loads in, you should find a board and information over how to play.
